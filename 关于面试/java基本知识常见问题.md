动态加载和静态加载的区别

静态加载是指在编译时确定对象类型，动态加载是指在运行时确定对象类型

反射是动态加载

Java反射是指在程序运行的环境中，对于任意一个类，都能知道这个类的属性和方法，对于一个对象，都能调用这个对象属性和方法，这种动态调用对象的方法叫做反射

反射的优点和缺点

动态加载，提高代码的灵活性

造成一点的安全问题，操作较慢



反射我们一般自己很少用，但是Java的反射机制是Java框架的灵魂
